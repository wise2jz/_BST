Introduction

This visualization demonstrates the step-by-step process of constructing a binary search tree from an array, as outlined in the lecture slides. The binary search tree follows these rules:

  • Each node can have at most 2 children (left and right)
  • Values less than the parent go to the left child
  • Values greater than the parent go to the right child
